# Create Interview Template

API cho phép Admin tạo mới một Interview Template (kịch bản phỏng vấn chuẩn).

## Endpoint
`POST /api/admin/templates`

## Authentication
Required (JWT Token) - Admin only

## Request Body (JSON)
```json
{
  "name": "Backend Junior - Laravel Focus",
  "description": "Standard interview template for Junior Backend Developer with Laravel expertise",
  "stack": "Backend",
  "level": "Junior",
  "duration_minutes": 60,
  "scoring_strategy": "weighted"
}
```

## Validation Rules
- `name`: required, string, max 255 characters
- `description`: optional, string
- `stack`: required, string (Backend, Frontend, Fullstack, DevOps, AI/ML, Mobile)
- `level`: required, string (Fresher, Junior, Middle, Senior, Tech Lead)
- `duration_minutes`: required, integer, min 1
- `scoring_strategy`: required, string, one of: `average`, `weighted`, `ai_only`

## Response Success (201 Created)
```json
{
  "success": true,
  "data": {
    "id": "9d2e5c8a-1234-5678-9abc-def012345678",
    "name": "Backend Junior - Laravel Focus",
    "status": "draft",
    "version": "v1"
  },
  "message": "Interview template created successfully."
}
```

## Business Rules
- Status mặc định: `draft`
- Version tự động bắt đầu từ `v1`
- Duration phải >= tổng time_limit của các câu hỏi (sẽ validate khi add questions)
- Template được tạo bởi Admin hiện tại (auto-tracked)

## Error Cases
- **400 Bad Request**: Validation failed hoặc missing required fields
- **401 Unauthorized**: Missing hoặc invalid token
- **403 Forbidden**: User không phải Admin

## cURL Example
```bash
curl -X POST http://localhost:8000/api/admin/templates \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Backend Junior - Laravel Focus",
    "description": "Standard interview template for Junior Backend Developer",
    "stack": "Backend",
    "level": "Junior",
    "duration_minutes": 60,
    "scoring_strategy": "weighted"
  }'
```
