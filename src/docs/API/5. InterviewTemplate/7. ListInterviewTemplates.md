# List Interview Templates

API để lấy danh sách Interview Templates với phân trang và bộ lọc.

## Endpoint
`GET /api/admin/templates`

## Authentication
Required (JWT Token) - Admin only

## Query Parameters
- `stack`: (optional) Filter theo stack (Backend, Frontend, etc.)
- `level`: (optional) Filter theo level (Junior, Senior, etc.)
- `status`: (optional) Filter theo status (draft, active, archived)
- `created_by`: (optional) Filter theo user ID của người tạo
- `sort_by`: (optional) Sort field, default: `created_at`
- `sort_order`: (optional) Sort direction (`asc`, `desc`), default: `desc`
- `per_page`: (optional) Số items per page, default: 15

## Filter Examples

### Filter by Stack and Level
```
GET /api/admin/templates?stack=Backend&level=Junior
```

### Filter by Status
```
GET /api/admin/templates?status=active
```

### Sort by Version (Descending)
```
GET /api/admin/templates?sort_by=version&sort_order=desc
```

### Pagination
```
GET /api/admin/templates?per_page=20&page=2
```

## Error Cases
- **401 Unauthorized**: Missing hoặc invalid token
- **403 Forbidden**: User không phải Admin

## cURL Example
```bash
curl -X GET "http://localhost:8000/api/admin/templates?stack=Backend&level=Junior&status=active&per_page=10" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```
