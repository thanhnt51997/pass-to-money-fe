# Update Interview Template

API cho phép Admin cập nhật metadata của Interview Template.

## Endpoint
`PATCH /api/admin/templates/{id}`

## Authentication
Required (JWT Token) - Admin only

## Request Body (JSON)
Partial update - chỉ gửi các field cần thay đổi:
```json
{
  "name": "Backend Junior - Updated Name",
  "description": "Updated description",
  "duration_minutes": 90,
  "scoring_strategy": "average"
}
```

## Validation Rules
- `name`: optional, string, max 255
- `description`: optional, string
- `stack`: optional, string
- `level`: optional, string
- `duration_minutes`: optional, integer, min 1
- `scoring_strategy`: optional, string, one of: `average`, `weighted`, `ai_only`

## Response Success (200 OK)
```json
{
  "success": true,
  "data": {
    "id": "9d2e5c8a-1234-5678-9abc-def012345678",
    "name": "Backend Junior - Updated Name",
    "status": "draft"
  },
  "message": "Template updated successfully."
}
```

## Business Rules
- **Không thể edit template đã archived**
- Version KHÔNG tự động tăng khi update metadata
- Chỉ update các field được gửi trong request (partial update)

## Error Cases
- **400 Bad Request**: 
  - Validation failed
  - Template đã bị archived
- **401 Unauthorized**: Missing hoặc invalid token
- **403 Forbidden**: User không phải Admin
- **404 Not Found**: Template không tồn tại

## cURL Example
```bash
curl -X PATCH http://localhost:8000/api/admin/templates/9d2e5c8a-1234-5678-9abc-def012345678 \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Backend Junior - Updated",
    "duration_minutes": 90
  }'
```
