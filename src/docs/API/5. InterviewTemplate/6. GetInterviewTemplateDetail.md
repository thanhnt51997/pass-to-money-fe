# Get Interview Template Detail

API để lấy chi tiết đầy đủ của một Interview Template, bao gồm danh sách câu hỏi đã được sắp xếp.

## Endpoint
`GET /api/admin/templates/{id}`

## Authentication
Required (JWT Token) - Admin only

## Request
No request body required.

## Response Fields
### Template Fields
- `id`: UUID của template
- `name`: Tên template
- `description`: Mô tả
- `stack`: Ngăn xếp công nghệ
- `level`: Cấp độ
- `duration_minutes`: Tổng thời gian (phút)
- `scoring_strategy`: Chiến lược chấm điểm
- `status`: Trạng thái (draft/active/archived)
- `version`: Phiên bản

### Question Fields (Ordered)
- `id`: UUID của template_question record
- `question_id`: UUID của question trong Question Bank
- `order`: Thứ tự hiển thị
- `weight`: Trọng số cho scoring
- `mandatory`: Bắt buộc hay không
- `time_limit`: Giới hạn thời gian (giây)
- `title`: Tiêu đề câu hỏi
- `type`: Loại câu hỏi

## Error Cases
- **401 Unauthorized**: Missing hoặc invalid token
- **403 Forbidden**: User không phải Admin
- **404 Not Found**: Template không tồn tại

## cURL Example
```bash
curl -X GET http://localhost:8000/api/admin/templates/9d2e5c8a-1234-5678-9abc-def012345678 \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```
