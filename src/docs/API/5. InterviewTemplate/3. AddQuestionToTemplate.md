# Add Question to Template

API cho phép Admin thêm một câu hỏi từ Question Bank vào Interview Template.

## Endpoint
`POST /api/admin/templates/{id}/questions`

## Authentication
Required (JWT Token) - Admin only

## Request Body (JSON)
```json
{
  "question_id": "8a1b2c3d-4567-89ab-cdef-0123456789ab",
  "order": 1,
  "weight": 2.5,
  "mandatory": true,
  "time_limit": 600
}
```

## Validation Rules
- `question_id`: required, string (UUID), must exist in Question Bank
- `order`: required, integer, min 0, must be unique per template
- `weight`: optional, numeric, min 0.01, default 1.0
- `mandatory`: optional, boolean, default true
- `time_limit`: optional, integer (seconds), min 1

## Response Success (200 OK)
```json
{
  "success": true,
  "message": "Question added to template successfully."
}
```

## Business Rules
- Question phải tồn tại trong Question Bank
- Question phải có status = `active`
- Order phải unique trong template (không trùng)
- Weight phải > 0
- Không thể add duplicate question vào cùng template

## Error Cases
- **400 Bad Request**: 
  - Validation failed
  - Question không tồn tại
  - Question không active
  - Weight <= 0
  - Order bị trùng
  - Question đã tồn tại trong template
- **401 Unauthorized**: Missing hoặc invalid token
- **403 Forbidden**: User không phải Admin
- **404 Not Found**: Template không tồn tại

## cURL Example
```bash
curl -X POST http://localhost:8000/api/admin/templates/9d2e5c8a-1234-5678-9abc-def012345678/questions \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "question_id": "8a1b2c3d-4567-89ab-cdef-0123456789ab",
    "order": 1,
    "weight": 2.5,
    "mandatory": true,
    "time_limit": 600
  }'
```
