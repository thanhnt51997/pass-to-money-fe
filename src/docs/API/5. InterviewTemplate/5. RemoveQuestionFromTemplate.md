# Remove Question from Template

API cho phép Admin xóa một câu hỏi khỏi Interview Template.

## Endpoint
`DELETE /api/admin/templates/{id}/questions/{questionId}`

## Authentication
Required (JWT Token) - Admin only

## Request
No request body required.

## Response Success (200 OK)
```json
{
  "success": true,
  "message": "Question removed from template successfully."
}
```

## Business Rules
- **Idempotent**: Gọi nhiều lần không gây lỗi
- Tự động recalculate total duration sau khi xóa
- Không ảnh hưởng đến Question trong Question Bank (chỉ xóa relationship)

## Error Cases
- **401 Unauthorized**: Missing hoặc invalid token
- **403 Forbidden**: User không phải Admin
- **404 Not Found**: Template không tồn tại

## cURL Example
```bash
curl -X DELETE http://localhost:8000/api/admin/templates/9d2e5c8a-1234-5678-9abc-def012345678/questions/8a1b2c3d-4567-89ab-cdef-0123456789ab \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

## Notes
- Nếu question không tồn tại trong template, API vẫn trả về success (idempotent)
- Sau khi xóa, nên reorder lại các câu hỏi còn lại nếu cần
