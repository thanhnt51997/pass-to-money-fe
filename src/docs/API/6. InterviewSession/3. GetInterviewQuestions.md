## Get Interview Questions

API để lấy danh sách câu hỏi của một phiên phỏng vấn (sử dụng snapshot đã được copy từ template khi tạo session).

## Endpoint
`GET /api/interviews/{id}/questions`

## Authentication
Required (JWT Token) - Candidate

## Path Parameters
- `id` (integer, required): ID của interview session.

## Business Rules
- **Ownership**: Chỉ owner (candidate đang đăng nhập) mới xem được câu hỏi.
- **Existing session**: Nếu session không tồn tại → `"Interview not found."`.
- **Permission**: Nếu session không thuộc về user hiện tại → `"You do not have permission to view these questions."`.
- **Nguồn dữ liệu**:
  - Câu hỏi được lấy từ snapshot (`interview_questions`) đã được tạo khi `CreateInterviewSession`.
  - Không truy vấn trực tiếp Question Bank để đảm bảo tính nhất quán.
- **Đáp án đúng**: API **không** trả về đáp án đúng, chỉ trả về nội dung câu hỏi và lựa chọn (nếu có).

> Lưu ý: Rule chặn theo trạng thái (`in_progress`) có thể được bật/tắt ở handler. Thiết kế chuẩn là chỉ hiển thị câu hỏi khi session đang `in_progress`.

## Response Success (200 OK)
```json
{
  "success": true,
  "data": {
    "interview_session_id": 14,
    "questions": [
      {
        "question_id": 101,
        "type": "choice",
        "content": "What is Laravel?",
        "options": ["Framework", "Library", "Language", "Database"],
        "order": 1
      },
      {
        "question_id": 102,
        "type": "theoretical",
        "content": "Explain Dependency Injection.",
        "options": null,
        "order": 2
      }
    ]
  },
  "message": "Questions retrieved successfully."
}
```

## Error Cases
- **400 Bad Request**:
  - `"Interview not found."`
  - `"You do not have permission to view these questions."`
  - (tuỳ cấu hình handler) `"Questions can only be retrieved for sessions that are in progress."`
- **401 Unauthorized**: Thiếu hoặc token không hợp lệ.
- **403 Forbidden**: Session không thuộc về user hiện tại.

## cURL Example
```bash
curl "http://localhost:8000/api/interviews/14/questions" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Accept: application/json"
```
