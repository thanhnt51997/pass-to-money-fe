## Submit Interview Answer

API để nộp hoặc cập nhật câu trả lời cho một câu hỏi trong phiên phỏng vấn.

## Endpoint
`POST /api/interviews/{id}/answers`

## Authentication
Required (JWT Token) - Candidate

## Path Parameters
- `id` (integer, required): ID của interview session.

## Request Body (JSON)
```json
{
  "question_id": 101,
  "answer_content": "Laravel là một PHP framework...",
  "time_spent": 120
}
```

### Fields
- `question_id` (integer, required): ID câu hỏi (snapshot) trong phiên phỏng vấn.
- `answer_content` (mixed, required):
  - Chuỗi cho câu hỏi tự luận/theoretical/essay/coding.
  - Mảng cho câu hỏi trắc nghiệm nhiều lựa chọn (MCQ).
  - Object cho các loại đặc biệt (ví dụ voice, file, v.v.).
- `time_spent` (integer, required): Thời gian (giây) user dùng để trả lời câu hỏi này (có thể cộng dồn ở FE).

## Business Rules
- **Existing session**: Nếu session không tồn tại → `"Interview not found."`.
- **Ownership**: User hiện tại phải là owner của session, nếu không → `"You do not have permission to answer for this interview."`.
- **Status**:
  - Chỉ cho phép submit khi session ở trạng thái `in_progress`.
  - Nếu không, trả về `"You can only submit answers for interviews that are in progress."`.
- **Question thuộc session**:
  - Hệ thống kiểm tra `question_id` có nằm trong danh sách questions của session hay không.
  - Nếu không, ném lỗi `"The specified question does not belong to this interview session."`.
- **Upsert**:
  - Nếu đã có answer cho `(interview_id, question_id)` → cập nhật `answer_content` và `time_spent`.
  - Nếu chưa có → tạo mới `InterviewAnswer`.
- **AI Evaluation**:
  - Với câu hỏi loại `theoretical`, `essay`, `coding`:
    - Answer được mark là `pending` cho AI.
    - Dispatch job `EvaluateSingleAnswerJob` để AI đánh giá trong background.

## Response Success (200 OK)
```json
{
  "success": true,
  "message": "Answer submitted successfully."
}
```

## Error Cases
- **400 Bad Request**:
  - `"Interview not found."`
  - `"You do not have permission to answer for this interview."`
  - `"You can only submit answers for interviews that are in progress."`
  - `"The specified question does not belong to this interview session."`
  - Validation thất bại (thiếu `question_id`, `answer_content`, `time_spent` không phải integer, ...)
- **401 Unauthorized**: Thiếu hoặc token không hợp lệ.
- **403 Forbidden**: Session không thuộc về user hiện tại.

## cURL Example
```bash
curl -X POST "http://localhost:8000/api/interviews/14/answers" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "question_id": 101,
    "answer_content": "Laravel là một PHP framework...",
    "time_spent": 120
  }'
```
