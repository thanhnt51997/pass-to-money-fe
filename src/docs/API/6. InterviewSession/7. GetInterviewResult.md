## Get Interview Result

API để lấy kết quả phỏng vấn sau khi candidate nộp bài và hệ thống/AI đã đánh giá.

## Endpoint
`GET /api/interviews/{id}/result`

## Authentication
Required (JWT Token) - Candidate

## Path Parameters
- `id` (integer, required): ID của interview session.

## Business Rules
- **Existing session**: Nếu session không tồn tại → `"Interview not found."`.
- **Ownership**: User hiện tại phải là owner của session, nếu không → `"You do not have permission to view this result."`.
- **Status hợp lệ**:
  - Chỉ cho phép xem kết quả nếu trạng thái session là `completed` (theo logic hiện tại trong handler).
  - Nếu chưa `completed` thì ném lỗi `"Results can only be viewed for completed interview sessions."`.
- **Trạng thái đánh giá AI**:
  - Nếu session đã `completed` nhưng AI chưa đánh giá xong:
    - API trả về:
      - `status: "processing"`
      - `interview_session_id`
  - Nếu AI đã đánh giá xong:
    - Trả về `score_total`, `score_detail`, `ai_summary`, `recommendation`.

> Lưu ý: Tên trạng thái domain và trạng thái business có thể được align thêm (ví dụ dùng `submitted` + cờ evaluated). Tài liệu này phản ánh đúng logic hiện tại trong handler `GetInterviewResultHandler`.

## Response Success – Đã được đánh giá
```json
{
  "success": true,
  "data": {
    "interview_session_id": 14,
    "score_total": 8.5,
    "score_detail": {
      "technical": 9.0,
      "communication": 8.0,
      "problem_solving": 8.5
    },
    "ai_summary": "Ứng viên có kiến thức nền tảng tốt về Laravel...",
    "recommendation": "hire"
  },
  "message": "Interview results retrieved successfully."
}
```

## Response Success – Đang xử lý (AI chưa xong)
```json
{
  "success": true,
  "data": {
    "interview_session_id": 14,
    "status": "processing"
  },
  "message": "Interview results retrieved successfully."
}
```

## Error Cases
- **400 Bad Request**:
  - `"Interview not found."`
  - `"You do not have permission to view this result."`
  - `"Results can only be viewed for completed interview sessions."`
- **401 Unauthorized**: Thiếu hoặc token không hợp lệ.
- **403 Forbidden**: Session không thuộc về user hiện tại.

## cURL Example
```bash
curl "http://localhost:8000/api/interviews/14/result" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Accept: application/json"
```
