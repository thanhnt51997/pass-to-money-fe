## Start Interview Session

API để bắt đầu một phiên phỏng vấn đã được tạo từ template.

## Endpoint
`POST /api/interviews/{id}/start`

## Authentication
Required (JWT Token) - Candidate

## Path Parameters
- `id` (integer, required): ID của interview session (trả về từ `CreateInterviewSession`).

## Request Body
Không có body. Chỉ cần path param và JWT token hợp lệ.

## Business Rules
- **Ownership**: Chỉ owner (candidate đang đăng nhập) của session mới được phép bắt đầu.
- **Existing session**: Nếu session không tồn tại → lỗi `"Interview not found."`.
- **Status hợp lệ**:
  - Chỉ cho phép bắt đầu nếu trạng thái hiện tại là `created`.
  - Sau khi bắt đầu, domain sẽ chuyển trạng thái sang `in_progress` và set `started_at` (thời gian hiện tại).
- **Không được start lại**: Nếu session đã `in_progress`, `submitted` hoặc `evaluated` thì không được start lại (sẽ ném exception từ domain).

## Response Success (200 OK)
```json
{
  "success": true,
  "data": {
    "interview_session_id": 14,
    "status": "in_progress",
    "started_at": "2026-01-28 10:00:00"
  },
  "message": "Interview session started successfully."
}
```

## Error Cases
- **400 Bad Request**:
  - `"Interview not found."`
  - `"You do not have permission to start this interview session."`
  - `"Interview session can only be started if status is created."` (từ domain)
  - `"Interview session has already been started."` (từ domain)
- **401 Unauthorized**: Thiếu hoặc token không hợp lệ.
- **403 Forbidden**: Session không thuộc về user hiện tại.

## cURL Example
```bash
curl -X POST "http://localhost:8000/api/interviews/14/start" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Accept: application/json"
```
