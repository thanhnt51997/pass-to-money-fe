# Update Question

API cho phép Admin cập nhật thông tin câu hỏi hiện có.

## Endpoint
`PATCH /api/admin/questions/{id}`

## Authentication
Required (JWT Token) - Admin only.

## Constraints
- Câu hỏi phải tồn tại trong hệ thống.
- Chỉ có thể chỉnh sửa nếu `status` khác `archived`.
- Hỗ trợ cập nhật từng phần (partial update).

## Request Body (JSON)
Các trường tương tự như Create Question, nhưng tất cả đều không bắt buộc.

```json
{
  "title": "Laravel Service Container (Updated)",
  "status": "active"
}
```

## Response Success (200 OK)
```json
{
  "success": true,
  "data": {
    "id": "uuid-v4-string",
    "title": "Laravel Service Container (Updated)",
    "status": "active"
  },
  "message": "Question updated successfully."
}
```

## Error Cases
- **400 Bad Request**: Câu hỏi không tồn tại hoặc đang ở trạng thái `archived`.
- **401 Unauthorized**: Missing or invalid token.
- **403 Forbidden**: User is not an admin.
