# Archive Question

API cho phép Admin lưu trữ (archive) câu hỏi thay vì xóa cứng.

## Endpoint
`DELETE /api/admin/questions/{id}`

## Authentication
Required (JWT Token) - Admin only.

## Rules
- Sử dụng Soft Delete.
- Các câu hỏi đã lưu trữ sẽ không xuất hiện trong việc chọn câu hỏi phỏng vấn.
- Thao tác có tính chất Idempotent (nếu đã lưu trữ thì trả về thành công).

## Response Success (200 OK)
```json
{
  "success": true,
  "message": "Question archived successfully."
}
```

## Error Cases
- **400 Bad Request**: Câu hỏi không tồn tại.
- **401 Unauthorized**: Missing or invalid token.
- **403 Forbidden**: User is not an admin.
