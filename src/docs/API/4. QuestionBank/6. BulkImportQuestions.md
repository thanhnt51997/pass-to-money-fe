# Bulk Import Questions

API cho phép Admin nhập hàng loạt câu hỏi phỏng vấn từ mảng JSON.

## Endpoint
`POST /api/admin/questions/bulk`

## Authentication
Required (JWT Token) - Admin only.

## Requirements
- Chấp nhận một mảng các đối tượng câu hỏi.
- Kiểm tra tính hợp lệ của từng mục một cách độc lập.
- Cho phép thành công một phần (partial success).
- Trả về tóm tắt quá trình nhập (số lượng thành công, lỗi chi tiết).

## Request Body (JSON)
```json
{
  "questions": [
    {
      "title": "Q1",
      "content": "Content 1",
      "type": "theoretical",
      "stack": "Backend"
    },
    {
      "title": "Q2",
      "content": "Content 2"
    }
  ]
}
```

## Response Success (200 OK)
```json
{
  "success": true,
  "data": {
    "total": 2,
    "success": 2,
    "errors": []
  },
  "message": "Bulk import process completed."
}
```

## Error Cases
- **400 Bad Request**: Định dạng dữ liệu không hợp lệ.
- **401 Unauthorized**: Missing or invalid token.
- **403 Forbidden**: User is not an admin.
