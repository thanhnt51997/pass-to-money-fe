# Create Question

Thiết kế API cho phép Admin tạo câu hỏi phỏng vấn mới trong ngân hàng câu hỏi.

## Endpoint
`POST /api/admin/questions`

## Authentication
Required (JWT Token) - Admin only.

## Request Body (JSON)
```json
{
  "title": "Laravel Service Container",
  "content": "Explain what is Service Container in Laravel and how it works.",
  "type": "theoretical",
  "stack": "Backend",
  "technologies": ["Laravel", "PHP"],
  "level": "Junior",
  "difficulty": "Medium",
  "expected_answer": "...",
  "evaluation_criteria": "Understanding of DI, IoC, Bindings...",
  "tags": ["core", "dependency-injection"]
}
```

## Validation Rules
- `title`: required, string, max 255.
- `content`: required, string.
- `type`: required, in: coding, theoretical, system_design, behavioral.
- `stack`: required, string.
- `level`: required, string.
- `difficulty`: required, string.
- `technologies`: optional, array.
- `tags`: optional, array.

## Response Success (201 Created)
```json
{
  "success": true,
  "data": {
    "id": "uuid-v4-string",
    "title": "Laravel Service Container",
    "status": "draft"
  },
  "message": "Question created successfully as draft."
}
```

## Error Cases
- **400 Bad Request**: Validation failed or missing required fields.
- **401 Unauthorized**: Missing or invalid token.
- **403 Forbidden**: User is not an admin.
