# AI Generate Question (Admin Tooling)

API cho phép Admin yêu cầu AI tạo các câu hỏi phỏng vấn dựa trên các tham số cấu hình.

## Endpoint
`POST /api/admin/questions/ai-generate`

## Authentication
Required (JWT Token) - Admin only.

## Inputs
- `stack` (string): Backend, Frontend...
- `level` (string): Junior, Senior...
- `technology` (string, optional): Laravel, React...
- `difficulty` (string): Easy, Medium, Hard.
- `type` (string): theoretical, coding...
- `count` (int, default 1): Số lượng câu hỏi muốn tạo (tối đa 10).

## Behavior
- Sử dụng AI để tạo các câu hỏi có cấu trúc.
- Trả về danh sách câu hỏi dự thảo (draft), chưa lưu trực tiếp vào cơ sở dữ liệu. Admin có thể xem xét trước khi lưu.

## Response Success (200 OK)
```json
{
  "success": true,
  "data": [
    {
      "title": "Laravel Middleware",
      "content": "What is middleware in Laravel and how do you create one?",
      "expected_answer": "...",
      "evaluation_criteria": "...",
      "tags": ["middleware", "laravel-core"]
    }
  ],
  "message": "AI questions generated successfully."
}
```

## Error Cases
- **400 Bad Request**: Định dạng dữ liệu không hợp lệ hoặc lỗi AI.
- **401 Unauthorized**: Missing or invalid token.
- **403 Forbidden**: User is not an admin.
