# List Questions

API phân trang cho phép Admin liệt kê và tìm kiếm các câu hỏi phỏng vấn.

## Endpoint
`GET /api/admin/questions`

## Authentication
Required (JWT Token) - Admin only.

## Query Parameters
- `stack` (string): Backend, Frontend...
- `technology` (string): Laravel, React...
- `level` (string): Junior, Senior...
- `difficulty` (string): Easy, Hard...
- `type` (string): theoretical, coding...
- `status` (string): draft, active, archived...
- `keyword` (string): Tìm kiếm trong tiêu đề hoặc tags.
- `per_page` (int): Số lượng mỗi trang (mặc định 15).
- `page` (int): Trang hiện tại.

## Response Success (200 OK)
```json
{
  "success": true,
  "data": {
    "items": [
      {
        "id": "uuid",
        "title": "Laravel Service Container",
        "type": "theoretical",
        "stack": "Backend",
        "level": "Junior",
        "difficulty": "Medium",
        "status": "active",
        "created_at": "2026-01-16 10:00:00"
      }
    ],
    "meta": {
      "total": 100,
      "current_page": 1,
      "last_page": 7
    }
  }
}
```

## Error Cases
- **401 Unauthorized**: Missing or invalid token.
- **403 Forbidden**: User is not an admin.
