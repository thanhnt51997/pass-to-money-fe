# Get Question Detail

API cho phép Admin lấy chi tiết đầy đủ của một câu hỏi phỏng vấn.

## Endpoint
`GET /api/admin/questions/{id}`

## Authentication
Required (JWT Token) - Admin only.

## Requirements
- Bao gồm tiêu chí đánh giá và câu trả lời mẫu.
- Bao gồm metadata (người tạo, thời gian tạo/cập nhật).

## Response Success (200 OK)
```json
{
  "success": true,
  "data": {
    "id": "uuid-v4-string",
    "title": "Laravel Service Container",
    "content": "...",
    "type": "theoretical",
    "stack": "Backend",
    "technologies": ["Laravel", "PHP"],
    "level": "Junior",
    "difficulty": "Medium",
    "expected_answer": "...",
    "evaluation_criteria": "...",
    "tags": ["core"],
    "status": "active",
    "created_by": 1,
    "created_at": "2026-01-16 10:00:00",
    "updated_at": "2026-01-16 10:30:00"
  }
}
```

## Error Cases
- **400 Bad Request**: Câu hỏi không tồn tại.
- **401 Unauthorized**: Missing or invalid token.
- **403 Forbidden**: User is not an admin.
