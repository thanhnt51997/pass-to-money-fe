# Get Levels

API công khai để lấy danh sách các cấp độ (Level) phỏng vấn có sẵn trong hệ thống.

## Endpoint
`GET /api/levels`

## Authentication
Not required (Public API)

## Request
No request body required.

## cURL Example
```bash
curl -X GET http://localhost:8000/api/levels \
  -H "Accept: application/json"
```

## Error Cases
None - This is a public endpoint that always returns available levels.
