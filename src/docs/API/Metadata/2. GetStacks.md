# Get Stacks

API công khai để lấy danh sách các ngăn xếp công nghệ (Stack) có sẵn trong hệ thống.

## Endpoint
`GET /api/stacks`

## Authentication
Not required (Public API)

## Request
No request body required.

## cURL Example
```bash
curl -X GET http://localhost:8000/api/stacks \
  -H "Accept: application/json"
```

## Error Cases
None - This is a public endpoint that always returns available stacks.
