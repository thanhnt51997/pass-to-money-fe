# Submit Interview Answer

Triển khai API nộp hoặc cập nhật câu trả lời cho một câu hỏi trong phiên phỏng vấn.

## Endpoint
`POST /api/interviews/{id}/answers`

## Authentication
Required (JWT Token)

## Request Body
- `question_id` (integer): ID của câu hỏi trong phiên phỏng vấn.
- `answer_content` (string | array): Nội dung câu trả lời (có thể là chuỗi hoặc mảng tùy loại câu hỏi).
- `time_spent` (integer): Thời gian thực hiện câu hỏi này (tính bằng giây).

## Business Rules
- Chỉ cho phép trả lời nếu trạng thái phiên phỏng vấn là `in_progress`.
- Nếu đã có câu trả lời cho câu hỏi này, hệ thống sẽ thực hiện cập nhật nội dung mới.
- Hệ thống kiểm tra tính hợp lệ của `question_id` để đảm bảo nó thuộc về phiên phỏng vấn hiện tại.

## Response Success
```json
{
    "success": true,
    "message": "Answer submitted successfully."
}
```

## Response Error
```json
{
    "success": false,
    "message": "You can only submit answers for interviews that are in progress."
}
```
