# Start Interview Session

Triển khai API bắt đầu phiên phỏng vấn.

## Endpoint
`POST /api/interviews/{id}/start`

## Authentication
Required (JWT Token)

## Parameters
- `id` (path): ID của phiên phỏng vấn.

## Business Rules
- Chỉ cho phép bắt đầu nếu trạng thái hiện tại là `pending`.
- Sau khi bắt đầu, trạng thái chuyển thành `in_progress`.
- Ghi nhận thời gian bắt đầu (`started_at`).
- Không cho phép bắt đầu lại phiên đã đang thực hiện hoặc đã hoàn thành.

## Response Success
```json
{
    "success": true,
    "data": {
        "interview_session_id": 1,
        "status": "in_progress",
        "started_at": "2026-01-16 16:00:00"
    },
    "message": "Interview session started successfully."
}
```

## Response Error
```json
{
    "success": false,
    "message": "Interview session can only be started if status is pending."
}
```
