# Submit Interview Session

Triển khai API nộp bài phỏng vấn.

## Endpoint
`POST /api/interviews/{id}/submit`

## Authentication
Required (JWT Token)

## Parameters
- `id` (path): ID của phiên phỏng vấn.

## Business Rules
- Chỉ cho phép nộp bài nếu trạng thái hiện tại là `in_progress`.
- Sau khi nộp bài:
    - Trạng thái chuyển thành `completed`.
    - Ghi nhận thời gian nộp bài (`submitted_at`).
- Không cho phép chỉnh sửa câu trả lời hoặc nộp lại sau khi đã hoàn thành.

## Response Success
```json
{
    "success": true,
    "data": {
        "interview_session_id": 1,
        "status": "completed",
        "submitted_at": "2026-01-16 16:20:00"
    },
    "message": "Interview session submitted successfully."
}
```

## Response Error
```json
{
    "success": false,
    "message": "Interview session can only be submitted if status is in_progress."
}
```
