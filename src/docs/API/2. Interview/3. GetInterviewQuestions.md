# Get Interview Questions

Triển khai API lấy danh sách câu hỏi của phiên phỏng vấn.

## Endpoint
`GET /api/interviews/{id}/questions`

## Authentication
Required (JWT Token)

## Parameters
- `id` (path): ID của phiên phỏng vấn.

## Business Rules
- Chỉ cho phép lấy danh sách câu hỏi nếu trạng thái phiên phỏng vấn là `in_progress`.
- Dữ liệu được lấy từ snapshot đã lưu trong cơ sở dữ liệu khi tạo phiên phỏng vấn, không truy vấn trực tiếp bảng câu hỏi gốc để đảm bảo tính nhất quán.
- Đáp án đúng của các câu hỏi sẽ không được tiết lộ trong API này.

## Response Success
```json
{
    "success": true,
    "data": {
        "interview_session_id": 1,
        "questions": [
            {
                "question_id": 101,
                "type": "choice",
                "content": "What is Laravel?",
                "options": ["Framework", "Library", "Language", "Database"],
                "order": 1
            },
            {
                "question_id": 102,
                "type": "text",
                "content": "Explain Dependency Injection.",
                "options": null,
                "order": 2
            }
        ]
    },
    "message": "Questions retrieved successfully."
}
```

## Response Error
```json
{
    "success": false,
    "message": "Questions can only be retrieved for sessions that are in progress."
}
```
