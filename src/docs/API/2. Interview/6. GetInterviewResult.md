# Get Interview Result

Triển khai API lấy kết quả phỏng vấn sau khi đã hoàn thành và được AI đánh giá.

## Endpoint
`GET /api/interviews/{id}/result`

## Authentication
Required (JWT Token)

## Parameters
- `id` (path): ID của phiên phỏng vấn.

## Business Rules
- Chỉ cho phép xem kết quả nếu trạng thái phiên phỏng vấn là `completed`.
- Nếu phiên phỏng vấn đã hoàn thành nhưng quá trình đánh giá AI chưa kết thúc, API sẽ trả về trạng thái `pending_evaluation`.
- Kết quả bao gồm tổng điểm, điểm chi tiết, nhận xét từ AI và đề xuất tuyển dụng.

## Response Success (Already Evaluated)
```json
{
    "success": true,
    "data": {
        "interview_session_id": 1,
        "score_total": 8.5,
        "score_detail": {
            "technical": 9.0,
            "communication": 8.0,
            "problem_solving": 8.5
        },
        "ai_summary": "Ứng viên có kiến thức nền tảng tốt về Laravel, trả lời lưu loát các câu hỏi về DI và Eloquent.",
        "recommendation": "hire"
    },
    "message": "Interview results retrieved successfully."
}
```

## Response Success (Pending Evaluation)
```json
{
    "success": true,
    "data": {
        "interview_session_id": 1,
        "status": "pending_evaluation"
    },
    "message": "Interview results retrieved successfully."
}
```

## Response Error
```json
{
    "success": false,
    "message": "Results can only be viewed for completed interview sessions."
}
```
