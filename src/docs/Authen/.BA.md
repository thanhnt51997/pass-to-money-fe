# Module: Authentication (Authen)

## Mục tiêu
Module Authentication chịu trách nhiệm xác thực và quản lý phiên đăng nhập của người dùng trong hệ thống Interview Platform.

Module này đảm bảo:
- Người dùng có thể đăng ký và đăng nhập an toàn
- Hệ thống cấp và quản lý JWT token
- Phân biệt người dùng đã xác thực và chưa xác thực
- Là nền tảng cho phân quyền (RBAC) ở các module khác

## Đối tượng sử dụng
- Guest (chưa đăng nhập)
- User (đã đăng nhập)
- Mentor
- Admin

## Phạm vi chức năng
Module Authen bao gồm các API sau:
- Đăng ký
- Đăng nhập
- Làm mới token
- Đăng xuất
- Lấy thông tin người dùng hiện tại

## Nguyên tắc thiết kế
- Sử dụng JWT cho xác thực
- Stateless API
- Token truyền qua Authorization Header
- Không xử lý logic nghiệp vụ ngoài xác thực
